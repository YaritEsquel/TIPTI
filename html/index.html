Guia básica de HTML

Introduccion

HTML (HyperText Markup Language) es el lenguaje standard para hacer pagina WEBS

en esta clase vamos aprender el uso de las etiquetas básicas como:

Heading <h1>: para titulos y subtitulos

Párrafo <p>: para párrafos

Div <div>: para agrupar contenidos.

Ejercicio

Vamos a realizar un ejercicio de practica en el cual cuenten lo que quieran en las instrucciones de abajo.

Sigan las instrucciones para realizarlo.

1- Crea un titulo principal utilizando una etiqueta<h1>

2- Agrega al menos dos titulos utilizando <h2> o <h3>

3- Escribe un nuevo párrafo utilizando la etiqueta <p>

4- Envuelve cada sección utilizando la etiqueta <div>
